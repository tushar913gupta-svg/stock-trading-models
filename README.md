Run the main.py file to see how the models will be used in a website setting.
Refer the buy_or_sell.ipnyb file in case you want to see my approach or you want to train a model on another set of stocks.
Run the test.py file if you want to see how the model performs in the past year.
Run the download.py file if you want to download any stock data not present in the Stock Data folder.

Models are trained on data from Jan 2022 to Jan 2025 and tested on data from from Jan 2025 to Dec 2025, achieving at least an 8-15 % ARR (depending on the stock type) for most stocks in the models' respective sectors and even higher returns over longer time periods.
